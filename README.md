## Overview       
* Created a Dashboard that shows the clean UI based on Input data stored. 
* Building an HR automation pipeline.     
* When someone uploads an Excel attendance file, your system processes it, 
flags late employees, counts their strikes, uses AI to write personalized warning emails, and auto-schedules a meeting on the 3rd strike.
